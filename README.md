# MagicBlock
使用OpenGL渲染三阶魔方,并实现自动打乱和求解的demo程序.

# 使用
鼠标拖拽 : 转动视角  
左键双击 : 开启/停止自动打乱  
右键双击 : 启动自动求解(静止状态)
